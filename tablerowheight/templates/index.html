TableRowHeight - Dynamically sizing DataTable row height to content

This demo uses a callback to size a DataTable row height to the data in a column.

Use the selector on the left to see the resizing in action. If the salary range you select includes salaries of $59,000 or more, heights are resized for a long error message, if you select a range that's less than $59,000 the row heights will resize to the shorter error messages.

There is no way I have found of dynamically resizing each individual row. It's all or nothing.
